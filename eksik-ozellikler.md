# Eksik Özellikler

- [ ] İkinci WhatsApp telefonu eklenecek.
- [ ] İkinci telefon verildiğinde projedeki ilgili tüm bileşenlerde (liste, detay, header ve CTA alanları) güncelleme yapılacak.
- [ ] Turhost hosting bilgileri firma tarafından iletilecek.
- [ ] Hosting bilgileri geldikten sonra GitHub `Actions Secrets` alanına şunlar eklenecek:
  - `TURHOST_FTP_SERVER`
  - `TURHOST_FTP_USERNAME`
  - `TURHOST_FTP_PASSWORD`
  - `TURHOST_SERVER_DIR`
- [ ] Secret'lar eklendikten sonra `Deploy Turhost` workflow'u çalıştırılıp canlı yayın doğrulanacak.
